지하철 예약 시스템
구현 내용
노선 및 도착 지점 데이터 처리
- 주요 도착 지점, 노선 정보를 XML 파일로 작성하여 시스템이 불러올 수 있도록 구현
- XML 파싱을 통해 노선 및 도착지 데이터를 프로그램에서 동적으로 로딩
- 탑승자 정보 입력 및 요금 계산

- 성인/청소년/아동 등 탑승자의 연령대나 신분 구분(학생, 장애인 등) 에 따라 요금 차등 적용
- 요금은 입력된 데이터에 따라 자동 계산

좌석 선택 기능 구현
- 실시간 좌석 정보를 제공하고 사용자가 원하는 좌석을 선택 가능
- 이미 예약된 좌석은 비활성화되어 선택 불가능

특수 차량 옵션 제공
- 조용한 칸, 휠체어 탑승 칸 등 사용자가 선호하는 차량 타입을   선택할 수 있도록 옵션 제공
- 선택한 차량에 따라 배정된 좌석 정보가 자동 갱신됨

결제 시스템 연동
- 카드사를 통한 간편 결제(시뮬레이션)
- 결제 완료 후 예약 정보 개별 xml에 저장

마이페이지 기능
- 사용자는 로그인 후 마이페이지에서 본인의 예약 이력 확인 가능
- 열차 이름, 도착 지점, 시간, 좌석 번호, 결제 금액 등의 상세 정보 제공
- 필요시 티켓 취소 또는 일정 변경 기능도 구현 가능

활용도
- 휠체어 이용자, 고령자, 임산부 등의 좌석 보장을 통한 교통 약자 배려 가능
- 승객의 예약 패턴 데이터를 활용하여 시간대별 운행 최적화 및 가격 정책 조정 가능
=>실제 혼잡 시간대 좌석 예약제 도입 시뮬레이션
